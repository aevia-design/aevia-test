# Plan 08-02 — Special Photo Upload Zones

## Objective
FP3 (Toy), FP4 (Steps), and FP5 (Art gallery) require separate special photo uploads that are distinct from the regular photo pool. Add named upload zones for each active functional page.

## Context
- @pages/template-engine.html
- @.planning/BRIEF.md — special photo types

## Special photo pools
```
FP3 Favourite toy:  1 photo — "toy photo" uploaded by customer → goes to Left slot
FP4 First steps:    1 photo — "steps photo" uploaded by customer → goes to Left slot
FP5 Art gallery:    1–2 photos — kid's artwork → goes to Left + Right slots
```
Right pages for FP3 and FP4 use regular photos from the main pool.
FP5 both pages use only artwork — artwork pool is separate from regular pool.

## Tasks

### Task 1 — Special upload panel
Below the main config bar, when any of FP3/FP4/FP5 are selected, show a "Special photos" section with one upload zone per active FP:

```
[🧸 Favourite toy photo: [Upload] filename.jpg ✓]
[👣 First steps photo:   [Upload] (not yet uploaded)]
[🎨 Art gallery (1-2):  [Upload] art1.jpg ✓  art2.jpg ✓]
```

Each upload zone:
- Label + file input button
- Preview thumbnail(s) after upload
- EXIF + orientation detection same as regular photos
- Stored in `window.specialPhotos.FP3`, `.FP4`, `.FP5` (arrays)

### Task 2 — Wire special photos to spread renderer
When rendering a spread that uses a special photo:
- FP3/FP4 left slot: use `window.specialPhotos.FP3[0]` / `.FP4[0]`
- FP5 left slot: use `window.specialPhotos.FP5[0]`
- FP5 right slot: use `window.specialPhotos.FP5[1]`
- If special photo not yet uploaded: show dashed placeholder with "Upload [type] photo above"

Special photos are NOT swappable via drag-and-drop from the main pool (they're fixed to their designated slot). They can only be replaced by uploading a new file in the special upload zone.

### Task 3 — Artwork orientation handling for FP5
FP5 has H and V variants (both pages). Pick variant based on the artwork's orientation:
- If artwork is landscape → use H variants for both pages
- If artwork is portrait → use V variants for both pages
- If 2 artworks uploaded with different orientations: use H for left, V for right (or V/H)? Default: match left artwork orientation.

## Verify
- Selecting FP3 shows "Favourite toy photo" upload zone
- Uploading a photo to it renders in FP3 left slot
- FP3 right slot uses next regular pool photo
- FP5 with 2 artwork uploads renders art-gallery spread correctly
- Un-uploaded special photo shows clear placeholder text
- Special upload zones only appear when corresponding FP is selected

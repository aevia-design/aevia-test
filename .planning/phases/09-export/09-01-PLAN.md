# Plan 09-01 — Export + Resolution Warnings + AI Captions

## Objective
Print-ready export via window.print() for the full book, resolution warnings on low-quality photos, and AI caption generation per slot (connecting to existing caption Cloud Function).

## Context
- @pages/template-engine.html (complete book from all prior phases)
- @pages/spread-preview.html — reference for print CSS and resolution warning
- @functions/caption/ — existing caption function

## Tasks

### Task 1 — Print CSS for full book
Add `<style media="print">` rules:
- Hide: config bar, photo strip, spread controls strip, sequence pill bar, all [Generate] buttons, spread labels, move/swap buttons
- Each `.spread-row` breaks to its own page (`page-break-after: always` or `break-after: page`)
- Each page canvas (600px) renders without margin/padding shifts
- SVG overlays print correctly (img tags print fine)
- Photo slots print with correct colors (no `-webkit-print-color-adjust` override needed for photos)
- "Print book" button at top and bottom of page → `window.print()`

### Task 2 — Resolution warnings
For each photo in the pool (regular + special):
- Minimum: 1500px on shortest side (300 DPI × 150mm slot = 1500px, consistent with spread-preview.html)
- Check after orientation detection
- Show warning badge on photo strip thumbnail: "⚠ Low res"
- In the spread render, if a low-res photo is placed in a slot, show a subtle yellow warning border on that slot
- Warning text: "Photo may print soft — shortest side {N}px, minimum 1500px"

### Task 3 — RAW format block
Same as spread-preview.html: detect .dng, .raw, .cr2, .nef, .arw extensions.
Block upload with alert: "RAW files are not supported. Please export as JPEG first."

### Task 4 — AI caption connection
Each `[✦ Generate]` button (from 07-03) calls:
```
POST https://europe-west1-aevia-uploads.cloudfunctions.net/generateCaption
Body: { imageDataUrl: base64string }
```
Or uses the existing caption API from spread-preview.html — check which endpoint works.
On success: fill caption div with response text.
Show spinner during call. Show "Caption error — try again" on failure (no hard block).

## Verify
- Print preview (Ctrl+P) shows clean book: no UI chrome, spreads paginated
- Low-res photo shows ⚠ badge in strip and yellow border in spread
- RAW upload attempt shows blocking alert
- AI generate button fills caption text for a loaded photo
- All verify steps from prior plans still pass (regression check)

# Plan 01-01 — Page Shell + Spec Data

## Objective
Create the `pages/spread-preview.html` skeleton and embed the spec data as a JS constant. After this plan the page loads in browser, shows a title, and the spec object is accessible in the console.

## Context
- Project conventions: plain HTML/CSS/JS, no frameworks, no build tools
- Asset paths from `pages/`: `../assets/`
- Spec source: `assets/Template/Spread 1 - test.xlsx` (already read — see data below)
- Local dev server: `npx serve . -p 8080` from project root

## Spec data (from xlsx — pre-converted, no runtime xlsx parsing needed)

```
Page variants for Spread 1:

L1 (page 1) — 1 slot, horizontal
  slot 1: center X=95, Y=100, W=150, H=100

L2 (page 3) — 1 slot, vertical
  slot 1: center X=95, Y=90, W=110, H=140

R1 (page 2) — 2 slots, both horizontal
  slot 1: center X=105, Y=55,  W=110, H=70
  slot 2: center X=105, Y=145, W=110, H=70

R2 (page 4) — 2 slots, both vertical
  slot 1: center X=65,  Y=100, W=70, H=100
  slot 2: center X=145, Y=100, W=70, H=100

R3 (page 5) — 2 slots, mixed
  slot 1: center X=50,  Y=60,  W=70,  H=100  (vertical)
  slot 2: center X=140, Y=132, W=105, H=70   (horizontal)

Page size: 200×200mm each. Bleed: 3mm (not in coordinates — code adds offset).
Coordinates = center of slot from top-left of page.
Convert to top-left: left = centerX - W/2, top = centerY - H/2
```

## Tasks

### Task 1 — Create `pages/spread-preview.html`
Create the HTML page with:
- Standard Aevia nav + footer (copy pattern from `pages/home.html`)
- A `<main>` section with heading "Spread Preview"
- A file upload area: input[type=file][multiple][accept="image/*"] labeled "Upload 3 photos"
- An empty `<div id="spread-container">` where the spread will render
- A `<script>` block containing the `SPREAD_SPEC` constant (see format below)
- Basic CSS: off-white background, centered layout, max-width 1200px

`SPREAD_SPEC` JS constant format:
```js
const SPREAD_SPEC = {
  pageSize: { w: 200, h: 200 }, // mm
  bleed: 3, // mm
  variants: {
    L1: { orientation: 'horizontal', slots: [{ cx: 95, cy: 100, w: 150, h: 100 }] },
    L2: { orientation: 'vertical',   slots: [{ cx: 95, cy: 90,  w: 110, h: 140 }] },
    R1: { slots: [{ cx: 105, cy: 55,  w: 110, h: 70 }, { cx: 105, cy: 145, w: 110, h: 70 }] },
    R2: { slots: [{ cx: 65,  cy: 100, w: 70,  h: 100 }, { cx: 145, cy: 100, w: 70,  h: 100 }] },
    R3: { slots: [{ cx: 50,  cy: 60,  w: 70,  h: 100 }, { cx: 140, cy: 132, w: 105, h: 70  }] },
  }
};
```

# Plan 06-03 — Full Book Scroll View

## Objective
Render all spreads in sequence as a vertically scrollable book canvas. Each spread row shows spread number, left + right pages, and basic controls.

## Context
- @pages/template-engine.html (with renderSpread from 06-01, sequence from 06-02)

## Tasks

### Task 1 — Spread row component
Each rendered spread gets a `.spread-row` wrapper with:
- Spread label: "Spread 1 (Standard)" / "Birthday wishes (Functional)"
- Left page + Right page side by side (each 600px canvas)
- For SP0: left side shows blank cover placeholder ("← Cover / Back")
- Controls strip below each spread (placeholders for 07-xx interactions):
  - `[↑ Move up]` `[↓ Move down]` `[Change spread type ▼]` buttons (disabled/stub for now)
- Spread index number displayed top-left (1, 2, 3...)

### Task 2 — Photo status bar
Above the book canvas, show:
- Total photos uploaded: N
- Photos assigned: N
- Photos unassigned: N (shown with warning color if > 0)
- Extra slots unfilled: N (shown with warning if > 0)

### Task 3 — Responsive layout
- At ≥ 1400px: left + right pages side by side (full spread)
- At < 1400px: left and right pages stacked vertically with a divider line
- Config bar stays sticky at top
- Photo strip stays below config bar, above book canvas

## Verify
- 20-page book renders 10 spread rows
- Each spread shows correct SVG overlay and photos
- SP0 shows right-only with blank left placeholder
- Photo status bar counts are correct
- Page scrolls smoothly through all spreads
- No layout overflow or horizontal scroll at 1440px

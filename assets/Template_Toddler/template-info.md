# Template: Toddler

A warm, playful photobook template for early childhood — newborns, toddlers, first years.

## Fonts

- **NT Comic** — primary display font
- **EB Garamond** — alternative / body text

## Caption style

Caption text uses **Plum** (`#493955`) as the font color.
